このページではよくある質問と対処法をまとめています。

## 全般

### Q. Scene ビューに制御点が表示されない

- Inspector の `ラティスエディターを開く` を押して `Lattice Tool` をアクティブにします。
- Scene ビュー右上のツールバーで `Lattice Tool` アイコンに切り替わっているか確認します。
- `Gizmos` トグルがオフになっていないか確認します。
- アクティブレイヤーがラティスレイヤー `[L]` になっているか確認します。ブラシレイヤー `[B]` が選択されている場合は制御点は表示されません。

### Q. プレビューを ON にしてもメッシュが更新されない

- Inspector の `メッシュ再構築オプション` を展開し、`(NDMF) Enable Lattice Preview` が ON の状態でハンドル操作を行っているか確認します。
- `変更後のグリッド分割数` を変更した場合は `適用` を押さないと制御点が再生成されません。
- Unity コンソールに NDMF 関連のエラーが出ている場合は、NDMF を 1.9.0 以降に更新してください。

### Q. 動作が重い

- ラティス分割数が高すぎると、制御点の数が増えて操作が重くなります。`変更後のグリッド分割数` を減らしてください。初期設定は `3, 3, 3` です。
- ブラシツールで `表面距離` を有効にしている場合、頂点数の多いメッシュでは計算に時間がかかることがあります。不要な場合はオフにしてください。
- `変位ヒートマップを表示` や `貫通表示` が有効な場合、Scene ビューの描画負荷が上がることがあります。

### Q. ビルドすると LatticeDeformer が消えてしまう

このコンポーネントはビルド時に自動削除されるため想定された挙動です。

- NDMF の Build Pipeline で `LatticeDeformer` は自動削除され、代わりにベイク済みメッシュ (`*_LatticeBaked`) が割り当てられます。
- Play モードを抜けると削除されたコンポーネントは自動的に復活します。

### Q. パッケージを更新したら UI が英語に戻った

Inspector の `ツールの言語` ドロップダウンで `日本語` を選択し直してください。ローカライズ設定は EditorPrefs に保存されますが、新しい Unity セッションでは英語に戻る場合があります。

## ラティスツール

### Q. 対称編集がミラーされない

- オーバーレイの `対称編集を有効化` がオンになっているかチェックします。
- 設定した `対称面` が正しいか確認します。色のついた面が設定した対称面になります。

### Q. 制御点をグローバル空間で移動できない

複数の制御点を選択している場合、グローバル空間での編集は無効になります。個別選択に切り替えるか、ローカル空間で操作してください。

## ブラシツール

### Q. ブラシで塗っても変形しない

- アクティブレイヤーがブラシレイヤー `[B]` になっているか確認します。
- `マスク` モードで頂点がマスクされていないか確認します。`マスクをクリア` でマスクをリセットできます。
- `ブラシ強度` が 0 に近い値になっていないか確認します。

### Q. ブラシがメッシュの裏側にも影響する

- `詳細設定` の `背面カリング` をオンにすると、カメラから見える面のみにブラシが適用されます。
- `表面距離` をオンにすると、メッシュ表面に沿った距離でブラシ範囲を計算するため、裏側への影響を防げます。

### Q. ブラシで別パーツのメッシュにまで影響する

`詳細設定` の `接続のみ` をオンにすると、ブラシ影響がメッシュのトポロジーで接続された頂点に限定されます。

## BlendShape

### Q. BlendShape 出力した変形が VRChat で動かない

- `BlendShape 名` が正しく設定されているか確認します。
- VRChat のアバターの Expression Parameters や Expression Menu で対応する BlendShape を制御しているか確認します。
- `テストモードに入る` で Unity 上で BlendShape が正しく動作するか事前に確認してください。

## ウェイト転送

### Q. ウェイト再計算後に関節の曲がりが不自然

- `最大転写距離` を大きくすると、より遠い元の頂点からもウェイトをコピーできるようになります。
- `法線角度閾値` を緩めると、より多くの頂点にウェイトが転写されます。
- Stage 2 の `補間を有効化` がオンになっていることを確認してください。

### Q. ウェイト再計算のオプションが表示されない

`ボーンウェイトを再計算` は `Skinned Mesh Renderer` を使用するメッシュでのみ利用可能です。`Static Mesh Source` を使用している場合は表示されません。
