Lattice Deformation Tool は、Unity 2022.3 以降で動作する VRChat 向け非破壊メッシュ変形エディタ拡張です。ラティスケージ変形とブラシ変形の 2 種類のレイヤーを組み合わせ、柔軟なメッシュ編集を実現します。

## 主な特徴

- **非破壊ワークフロー**: NDMF (Non-Destructive Modular Framework) と連携し、元メッシュを書き換えずに変形を適用・プレビューできます。ビルド時に自動ベイクされ、コンポーネントは削除されます。
- **レイヤーとグループ**: 複数のラティスレイヤー・ブラシレイヤーを重ねて変形を管理できます。グループ単位で独立した変形セットを構成し、各レイヤーにはウェイト (0〜1) や有効/無効の切り替えが可能です。
- **ラティスツール**: Scene ビュー上でラティスケージの制御点を直接操作してメッシュを変形します。対称編集、境界のみ/全制御点の切り替え、補間方式 (Trilinear / Cubic Bernstein) の選択に対応します。
- **ブラシツール**: ブラシで直感的にメッシュを塗るように変形できます。法線方向の押し出し、移動、スムージング、マスクの 4 つのモードを搭載しています。
- **頂点選択ツール**: ブラシレイヤー上で頂点を直接選択し、移動・回転・スケールで変形します。プロポーショナル編集にも対応しています。
- **BlendShape 出力**: グループ単位で変形結果を BlendShape (ShapeKey) として出力できます。アニメーションカーブで補間を制御し、テストモードでリアルタイムプレビューが可能です。
- **BlendShape インポート**: 既存メッシュの BlendShape をブラシレイヤーとして読み込み、さらに編集できます。
- **左右操作 (L/R Operations)**: レイヤーの変形を軸で分割 (左を残す / 右を残す) したり、軸反転でミラーリングしたりできます。VRChat アバターの左右対称ワークフローに便利です。
- **ボーンウェイト再計算**: 大きな変形後にボーンウェイトを自動的に再計算します。転写できなかった頂点も周囲から滑らかに補間されるため、変形後も自然な関節の動きが保たれます。
- **5 言語対応**: English / 日本語 / 한국어 / 简体中文 / 繁體中文

## 必要環境の要約

- Unity 2022.3 LTS 以降 (VRChat Creator Companion 推奨バージョン)
- NDMF 1.9.0 以上

## ドキュメント構成

- **前提条件**: 必要な環境と互換性
- **クイックスタート**: `LatticeDeformer` の基本設定から NDMF ビルドまでの手順
- **ガイド**: レイヤーとグループ、ブラシツール、頂点選択ツール、BlendShape、左右操作、ウェイト転送の詳細解説
- **トラブルシュート**: よくある症状と確認ポイント
- **リファレンス**: Inspector と Scene ツールの全設定項目の説明

## 用語と表記について

- Unity と VCC の UI は日本語表示を基準に記載し、必要に応じて英語表記 (カッコつき) を併記します。
- Lattice Deformation Tool 本体の一部項目は日本語 UI でも英語のまま表示されます。本書でも実際のラベルに合わせた表記を使用します。
