<Icon name="mirror" alt="ミラー" /> VRChat アバターの左右対称ワークフローを効率化するための機能です。レイヤーの変形データを軸で分割したり、反転コピーしたりできます。

## 左右操作 (L/R Operations)

Inspector のレイヤー設定に `左右操作` セクションがあります。

### 分割 (Split)

レイヤーの変形データを軸で 2 分割し、片側のみ残します。

| ボタン | 説明 |
| --- | --- |
| `左を残す` (Split L) | X 軸正側 (右側) の変形をゼロにし、左側の変形のみ残します |
| `右を残す` (Split R) | X 軸負側 (左側) の変形をゼロにし、右側の変形のみ残します |

### 反転 (Flip)

レイヤーの変形データを軸を基準にミラーリングします。

| ボタン | 説明 |
| --- | --- |
| `X反転` (Flip X) | X 軸を基準に全変形データをミラーリング |
| `Y反転` (Flip Y) | Y 軸を基準に全変形データをミラーリング |
| `Z反転` (Flip Z) | Z 軸を基準に全変形データをミラーリング |

## 典型的なワークフロー

VRChat アバターで片側だけ変形を作り、反対側にミラーリングする手順です。

### 片側編集 → 反転コピー

1. ラティスまたはブラシで片側 (例: 右半分) を変形する
2. `レイヤーを複製` で複製を作成する
3. 元のレイヤーで `右を残す` を実行 (右半分のみ残る)
4. 複製したレイヤーで `左を残す` を実行 (左半分のみ残る)
5. 複製したレイヤーで `X反転` を実行 (左半分の変形を右側にミラー)

これで左右対称の変形が完成します。

### 対称編集の活用

上記の手動操作の代わりに、各ツールの**対称編集**機能を使うと、編集中にリアルタイムで対称軸の反対側にも同時に変形が適用されます。

- **ラティスツール**: オーバーレイの <Icon name="mirror" /> `対称編集を有効化` → `対称モード` (`ミラー` / `反対称`) と `対称軸` を設定
- **ブラシツール**: オーバーレイの <Icon name="mirror" /> `ミラーを有効化` → `ミラー軸` を設定

:::tip
対称編集を使ってリアルタイムに左右対称に編集するのが最も効率的です。左右操作 (Split/Flip) は、片側だけ変形を作った後や、既存の変形を反転したい場合に使うと便利です。
:::
